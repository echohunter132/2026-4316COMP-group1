# 2026-4316COMP-Group1

## Project Overview
This project analyses Covid-19 Cases, Deaths and Recoveries using Python to extract insights and produce visualisations.

## Folder Structure
- data/ : datasets
- src/ : main Python code
- visualisations/ : individual graphs
- docs/ : reports and meeting minutes

## How to Run
### 1. Install dependencies
pip install -r requirements.txt
### 2. Run the Program
python src/main.py
